YandexMusicDesktopWidget — установка
=====================================

1. Перетащите «YandexMusicDesktopWidget.app» в папку «Программы» (Applications).

2. Первый запуск (ВАЖНО):
   Приложение распространяется без платной подписи Apple, поэтому macOS
   при первом запуске покажет предупреждение «не удаётся проверить разработчика».

   Сделайте так:
   • Откройте Терминал и выполните одну команду (снимает «карантин»):

       xattr -dr com.apple.quarantine /Applications/YandexMusicDesktopWidget.app

   • Затем запустите приложение обычным двойным кликом.

   (Альтернатива без терминала: правый клик по приложению → «Открыть» →
    в диалоге снова «Открыть». Если не помогает — используйте команду выше.)

3. Дайте разрешение «Универсальный доступ» (Accessibility):
   Системные настройки → Конфиденциальность и безопасность → Универсальный доступ →
   включите YandexMusicDesktopWidget. Это нужно для чтения трека и лайков.

4. Войдите в Яндекс (в окне приложения → Настройки) — для обложек HD и лайков.

5. Добавьте виджет на рабочий стол: правый клик по столу → «Изменить виджеты» →
   найдите YandexMusicWidget.

Готово — приятного прослушивания!
