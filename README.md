# YandexMusicDesktopWidget

Виджет рабочего стола и меню-бар для macOS, который показывает текущий трек из
**Яндекс Музыки** — с обложкой в HD, лайками и управлением воспроизведением.

> macOS 14+ · бесплатно

## Возможности

- 🎵 Виджеты трёх размеров на рабочем столе
- 🖼 Обложка в высоком качестве
- ❤️ Лайк / дизлайк прямо из виджета
- ⏯ Плей/пауза, переключение треков, перемотка
- 🪟 Меню-бар попап (два стиля) и окно приложения
- 🌍 Русский и английский интерфейс
- 🚀 Автозапуск при входе в систему

## Установка

1. Скачайте `YandexMusicDesktopWidget-1.0.0.dmg` из раздела **[Releases](../../releases)**.
2. Откройте DMG и перетащите приложение в папку **Программы**.
3. При первом запуске macOS предупредит о неизвестном разработчике. Выполните в Терминале:
   ```bash
   xattr -dr com.apple.quarantine /Applications/YandexMusicDesktopWidget.app
   ```
   затем запустите приложение двойным кликом.
4. В окне приложения → **Настройки** войдите в Яндекс (для обложек и лайков) и
   включите **Универсальный доступ**, если попросит.
5. Добавьте виджет: правый клик по рабочему столу → **Изменить виджеты** → найдите
   «YandexMusicWidget».

> Приложение подписано ad-hoc (без платного Apple Developer), поэтому шаг со
> снятием «карантина» нужен один раз — это нормально для бесплатных macOS-приложений.
